# Gift-Shop-Sales-Analysis-Dashboard-using-Excel
Excel dashboard for analyzing gift shop sales across months, occasions, categories, and cities.

## Project Overview
This project analyzes the sales performance of a gift shop using Microsoft Excel.  
The dashboard provides insights into revenue trends, top-selling products, city-wise sales, category distribution, and occasion-based performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Advanced Excel formulas

## Dashboard Highlights
- Total Revenue, Average Revenue, Total Orders, Delivery Time KPIs
- Month-wise revenue and quantity trends
- Top 5 products by sales
- Top 10 cities by revenue
- Occasion-wise sales analysis
- Category-wise product count
- Interactive slicers for Occasion, Day, and Category

## Dataset
The dataset contains:
- Customers

## Key Insights
- Peak sales observed during **Anniversary and Raksha Bandhan**
- Highest revenue generated in **February and August**
- Cakes and Colors are among the top-selling categories
- Certain cities consistently outperform others in sales

## Conclusion
This Excel dashboard enables stakeholders to quickly understand sales performance and make informed, data-driven decisions using interactive visualizations.
